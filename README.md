# 🥗 Калькулятор БЖУ для Telegram

Простой и мотивирующий Telegram-бот для расчёта суточной нормы **белков, жиров и углеводов (БЖУ)** на основе вашего веса и цели.  
Работает **и для мужчин, и для женщин**.  
Всё, что нужно — ответить на три вопроса!

---

## ✨ Возможности

- 📊 **Рассчитывает БЖУ по простой и эффективной схеме**:
  - **Жиры** = вес × 1 г
  - **Белки**:
    - Похудение → вес × 2.2 г
    - Набор / Поддержание → вес × 2.0 г
  - **Углеводы**:
    - Похудение → вес × 2 г
    - Поддержание → вес × 3 г
    - Набор → вес × 4 г
- 🔁 **Кнопка «🔄 Перерасчёт»** — начать заново можно на любом этапе (кроме первого).
- 💬 Удобный пошаговый диалог: пол → вес → цель.
- 🏆 Вдохновляющее сообщение в конце:  
  > «Ты уже Чемпион! Стань лучшей версией себя с дисциплиной 💪»

## 🚀 Как запустить бота (для разработчиков)

### 1. Получите токен бота
1. Напишите [@BotFather](https://t.me/BotFather) в Telegram.
2. Создайте нового бота командой `/newbot`.
3. Скопируйте выданный **токен** (выглядит как `123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ`).

### 2. Установите зависимости
Убедитесь, что у вас установлен Python 3.7+.

```bash
pip install python-telegram-bot python-dotenv
```

### 3. Настройте токен
Создайте файл `.env` в корне проекта:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

Пример:
```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ
```

### 4. Запустите бота
```bash
python bot.py
```

Вы увидите сообщение:  
✅ `Бот запущен!`

Теперь вы можете писать ему в Telegram!

## 📱 Как пользоваться (для пользователей)

1. Найдите бота в Telegram (если знаете его имя) или разверните своё собственное.
2. Нажмите **/start**.
3. Следуйте инструкциям:
   - Выберите пол
   - Введите вес (например: `70.5`)
   - Выберите цель: **Похудение**, **Поддержка** или **Набор**
4. Получите готовый расчёт БЖУ и калорий!
5. Хотите пересчитать? Нажмите **«🔄 Перерасчёт»** в любой момент 💡

## 💡 Советы по использованию

- Расчёт ориентировочный — корректируйте под самочувствие и результаты.
- При похудении следите, чтобы калории не падали ниже безопасного уровня (~1200 ккал для женщин, ~1500 для мужчин).
- Бот не сохраняет ваши данные — конфиденциальность гарантирована!
